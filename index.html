My Projects

These are some of the projects I've made. Check them out here!

PIEMAP

Coded in C++, PIEMAP is a GIS Mapper Application that is complete with a functional user interface, and pathfinding algorithms. Due to academic integrity restrictions, code is not fully public.

Poetify

Poetify is a Machine Learning Model that transforms internet reviews into stylized poems. It does so by taking a review as an input, summarizes the review, and generates poems using the shortened summary.

OpenCV Virtual Whiteboard

Virtual Whiteboard is a computer vision project that enables users to draw using a computer webcam. This project uses the Opencv python library and Mediapipe. Users can use index and middle finger to select colors, and draw with the index finger.

ReactJS Video Player

The ReactJS Video Player is a web application that uses the YouTube API to search for and play back videos. I built this as a way to practice using the ReactJS framework.

Indago

Indago is a discord bot which can be used by students to assist in studying, with flashcard capabilities, music, videos, and workload scheduling commands.

Nonstop

Nonstop is a Traffic control algorithm simulation which allows driverless cars to slow down and speed up accordingly, if cars are approaching an intersection. Used to avoid collision with other vehicles and pedestrians.
